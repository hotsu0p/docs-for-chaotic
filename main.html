Moderation

Verification System

With Chaotic you can create a verification system easily

1- setverificationchannel #channel
2- setverificationrole @role
3- setverificationmessage < message >

Done!

Auto Role

This will automatically add the specified role every time a member joins the guild

Lock and Unlock Channel

This will easily lock the channel provided to stop grief, as you can unlock it anytime

Advanced Warning

When warning a user, you can use the Auto Kick Command which will auto kick every member with a certain number of warns, as you can also clear warns and check warn logs with its date and reason.

Advanced Logging

This feature includes:

  • Deleted message logging
  • Nickname logging
  • Edited messages logging
  • Member logging
  • Role logging

Disable Mentions

If enabled, this feature will delete every ping and automatically then warn the user that pinged you. This command is automatically disabled once you send any message again in any channel

Leave & Welcome Message

Set up both welcome and leave messages to your server easily

Ban & Soft Ban

Ban a member from the server, then unban directly. This will delete all their messages

Purge

Delete the specified amount of messages from the provided channel, this feature can also be used with the command Warn Purge

Other Commands

These commands include: setnickname, slowmode, mute, addrole, removerole
